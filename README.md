# Red Cogs?  In this economy?

Yeah, its time for some cogs.  Some are hyper personal to me and my local area, others not so much.  Feel free to use them as a base for your own cog, or don't.

## Requirements

The current requirements are beautifulsoup4 and aiohttp.  There can be added with

```
[p]pipinstall beautifulsoup4
```
```
[p]pipinstall aiohttp
```
